Limpieza de Datos¶

Una vez importados los datos, explorados casos particulares y habiendo tenido un primer acercamiento visual al conjunto, el siguiente paso es prepararlo para realizar un anlisis ms sofisticado, como lo es una Regresin Estadstica, una divisin en subgrupos a partir de caractersticas particulares, o simplemente obtener un resumen ms detallados de los datos.

Para todos los casos anteriores, lo primero que se necesita es lo que el estadista neozelands Hadley Wickham define como Tidy Data; la idea general de este concepto se centra en convertir los datos de cualquier estructura en la que fueron recibidos a una que permita una manipulacin sencilla y estandarizada, y que permita directamente introducirla a una herramienta de anlisis con el mismo nivel de orden.

Wickham destaca tres caractersticas fundamentales que poseen el Tidy Data:

  1. Cada variable forma una sola columna
  2. Cada observacin forma una fila
  3. Cada tipo de unidad de observacin forma una tabla

Para mayor informacin sobre el concepto de Tidy Data, puede consultarse el Artculo Acadmico original (de Acceso Libre), as como el Repositorio Plico asociado a l.
